# GenZ Fashion

A modern, responsive fashion e-commerce website showcasing winter jacket collections for men and women.

## Live Demo

**Visit the live site:** [https://saifulislamrumi.github.io/Genz-fashion-Rumi/](https://saifulislamrumi.github.io/Genz-fashion-Rumi/)

## About

GenZ Fashion is a stylish e-commerce platform designed for the Gen-Z audience, featuring a curated collection of trendy winter jackets. The website offers a clean, modern interface with easy navigation and engaging product displays.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean and intuitive interface with smooth navigation
- **Product Showcase** - Separate sections for women's and men's jacket collections
- **Hero Section** - Eye-catching landing area with call-to-action
- **Product Cards** - Detailed product display with images, descriptions, and pricing
- **Footer Section** - Informative cards highlighting key services (Perfect Fit, Free Exchanges, Customer Support)

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/saifulislamrumi/Genz-fashion-Rumi.git
```

2. Navigate to the project directory:
```bash
cd Genz-fashion-Rumi
```

3. Open `index.html` in your browser:
```bash
open index.html
```
Or simply double-click the `index.html` file.
